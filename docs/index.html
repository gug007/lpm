Local Project Manager

Start, stop, and switch between dev projects with one command. CLI + native macOS desktop app.

Download View on GitHub

Install CLI

curl -fsSL https://raw.githubusercontent.com/gug007/lpm/main/install.sh | bash

Features

One command to start

Define your services in YAML, launch everything with lpm myapp.

Auto-detect

Detects Rails, Next.js, Go, Django, Flask, Docker Compose automatically.

Fast switching

Switch between projects instantly. Stop one, start another in a single command.

Profiles

Run subsets of services. Start just the API, or the full stack.

Desktop app

Native macOS app with live terminal, config editor, and theme support.

Any stack

If it runs in a terminal, lpm can manage it. No Docker required.

Commands

lpm <project> Start in background
lpm start <project> Start and open terminal
lpm switch <project> Stop all, start another
lpm kill [project] Stop a project (all if none given)
lpm init [name] Create config from current directory
lpm edit <project> Open config in $EDITOR
lpm list List all projects
lpm status <project> Show project details
